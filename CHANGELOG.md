# Changelog

## [Unreleased]

### Added
- **15 言語対応** — String Catalog (.xcstrings) による多言語ローカリゼーション
  - 英語 (メイン), 日本語, 簡体中文, 繁体中文, 韓国語, フランス語, ドイツ語, スペイン語, ポルトガル語, イタリア語, ロシア語, タイ語, インドネシア語, ベトナム語, アラビア語
- **使い方ガイドのリッチ化**
  - ヘッダー演出 (アプリアイコン + キャッチコピー)
  - セクション折りたたみ (アコーディオン式)
  - SF Symbols バウンスアニメーション
  - スクロール連動フェードインアニメーション
  - タイムライン風ステップ表示 (番号間の接続線)
- **ウィジェット追加ガイドを 2 セクションに分割**
  - 「アプリアイコンからすばやく追加」(2 ステップ)
  - 「ホーム画面からウィジェットを追加」(4 ステップ)
- **コピー完了トースト** — コピーボタン押下時に「クリップボードにコピーしました」をアニメーション表示
- **バックアップ機能** — メモの状態を手動保存・復元 (半モーダルシート)
- **バックアップの自動保存** — 復元時に現在の状態を自動でバックアップ
- **データ削除にバックアップを含む** — ボード個別 / 全ボード削除でバックアップも一括削除
- **ボード名リセット** — ボード個別削除でボード名もデフォルトに初期化
- **削除アラートの改善** — バックアップも消える旨を明記
- **Undo / Redo** — 手動スタックベースのテキスト編集取り消し・やり直し
- **コピーボタン** — ツールバー右上にクリップボードコピーボタン
- **使い方ガイド** — questionmark.circle ボタンから半モーダルで表示
- **すべてのデータを削除** — 設定画面に全ボード一括削除ボタン
- **Apple Watch アプリ** — iPhone のメモを Watch で閲覧
- **UI テスト 25 件** — ツールバー (8), スナップショットシート (8), Undo/Redo (9)
- **ユニットテスト 78 件** — MemoStore, BoardManager, SnapshotStore, TextUndoManager, WatchConnectivity

### Fixed
- 絵文字の文字化け — ヘルプガイド内の 🕐📋⚙ を日本語テキストに置換
- UI テストの状態依存 — テスト順序に依存しないテスト設計に修正
